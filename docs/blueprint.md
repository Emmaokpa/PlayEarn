# **App Name**: RewardPlay

## Core Features:

- Game Display: Display games from Playgama embedded via iframe.
- Ad Integration: Integrate video ads that users can watch to earn rewards.
- Reward System: Track user rewards based on game play and ad views.
- User Authentication: Basic user authentication to manage individual progress and rewards.
- Reward Redemption: Allow users to redeem accumulated rewards.
- Progress saving: Track the users progression in each of the offered games

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to reflect trust and engagement.
- Background color: Dark purple (#212130) to ensure content clarity.
- Accent color: Energetic green (#90EE90) to highlight rewards and calls to action.
- Body and headline font: 'PT Sans', a versatile sans-serif for a modern yet approachable feel.
- Use clean, minimalistic icons to represent game categories, rewards, and user profile sections.
- Design a user-friendly layout optimized for Telegram Mini Apps, ensuring easy navigation and clear information hierarchy.
- Incorporate subtle animations on user interactions, such as reward confirmations, to enhance engagement.
